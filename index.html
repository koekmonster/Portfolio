Projects

In 2024, I had the privilege of leading a team for the Networking module Semester Project, CMPG 315. Despite initial challenges, our project achieved a perfect score. We developed an HMS (Human Movement Sciences) Data Platform with several key features and functionalities.

Project Overview

The HMS Data Platform includes large file storage, video streaming, access controls, user isolation, and video compression. It provides a secure and efficient environment for managing assignments and feedback.

Key Features

  • Secure Login
  • Create, Edit, and View Assignments
  • Compress and Upload Videos for Submissions
  • View Submissions
  • Stream Videos
  • Provide & View Feedback on Videos
  • Download Feedback & Marks in CSV/XLSX
  • Large File Storage
  • User Administration

Supported Environments

  • Mobile Applications
  • Desktop Optimized Web Application

User Roles

  • Administrator: Unrestricted System Access
  • Lecturer: Create Assignments, View Submissions, Stream Videos, Provide Feedback
  • Student: View Assignments, Make Submissions, View Feedback

Impact

The platform is designed to enhance customer satisfaction by providing a comprehensive solution for managing educational content and feedback in a secure and user-friendly manner.
